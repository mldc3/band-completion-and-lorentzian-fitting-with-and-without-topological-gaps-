# Assets Index

This file lists the selected figures that should be included in the polished repository. The images are currently provided as flat files so they can be uploaded easily. Copilot should move them into the final `assets/figures/` folders.

## Selected figures

| File | Suggested final folder | Why it should be included |
|---|---|---|
| `fig_disorder_8_large_config1.png` | `assets/figures/disorder/` | Compact example of a non-periodic 8-large-cylinder configuration with measured peaks and reconstructed bands. Good first visual for the README. |
| `fig_defect_15_large_1_medium.png` | `assets/figures/defects/` | Shows an on-site geometric defect: one medium cylinder inserted into a large-cylinder chain. Useful to distinguish defect physics from SSH topology. |
| `fig_disorder_16_medium_config1.png` | `assets/figures/disorder/` | Longer medium-cylinder chain. Demonstrates missing modes, higher mode density, and why band completion is needed. |
| `fig_weak_coupling_10_large_4mm_interleaved.png` | `assets/figures/weak_coupling/` | Shows the difficulty of working with very weak 4 mm diaphragm couplings. Important for explaining experimental limitations. |
| `fig_disorder_16_large_config1.png` | `assets/figures/disorder/` | Long large-cylinder disordered case. Useful comparison against the medium-cylinder version. |
| `fig_topological_24_medium_large_medium.png` | `assets/figures/topological/` | Strong visual for SSH-like alternating couplings in a 24-cylinder chain. Should be one of the main README figures. |
| `fig_topological_24_medium_small_large.png` | `assets/figures/topological/` | Strong-coupling-contrast topological configuration. Good for discussing gap opening and weak/strong hopping. |
| `fig_topological_7_large_central_mode.png` | `assets/figures/topological/` | Short pedagogical example of a central localized/gap-related mode. Useful in theory and results sections. |
| `fig_temperature_16_medium_cold_topological.png` | `assets/figures/temperature/` | Shows temperature as a perturbation. Important for explaining why not every isolated peak is topological. |
| `fig_comparison_two_large_cylinders_diaphragm_size.png` | `assets/figures/comparisons/` | Clean comparison of diaphragm aperture effects using a simple two-cylinder setup. Useful for explaining coupling calibration. |

## Recommended README visual order

1. `fig_topological_24_medium_large_medium.png`
2. `fig_disorder_8_large_config1.png`
3. `fig_topological_7_large_central_mode.png`
4. `fig_temperature_16_medium_cold_topological.png`

This order tells a good story: strong final result, baseline/disorder, topological candidate, and careful limitation.
