# Optional Extra Figures to Capture Later

The current ZIP already includes a strong set of figures. If you want the repository to look even more complete, the following additional captures would be useful.

## Highest value additions

1. **A clean homogeneous baseline**  
   Capture a page showing a simple periodic chain, such as 8 large cylinders with all diaphragms equal. This would make the transition from periodic bands to SSH-like gaps clearer.

2. **A no-diaphragm case**  
   Capture a page where the cylinders are not coupled by diaphragms. This would visually prove why coupling is necessary for band formation.

3. **A clear comparison of small, medium, and large diaphragms**  
   Capture the best page where diaphragm aperture is compared directly. This is useful for explaining that aperture controls the effective hopping.

4. **A figure where measured and interpolated points are clearly distinguished**  
   This would help explain the band-completion algorithm.

5. **A clean SSH-theory schematic if available**  
   A diagram of alternating strong/weak links would be excellent for the theory page.
